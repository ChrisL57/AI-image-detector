# AI-image-detector
Détecteur d'image IA

Un outil pour analyser les images générées par IA essentiellement SD, SDXL

## Fonctionnalités
- Lecture des métadonnées (Automatic1111, ComfyUI, Fooocus)
- Détection de signatures spécifiques (Pony V6, Illustrious)
- Analyse visuelle via IA
